SEIS Aftershock Models — Evaluation Report

4 model families (XGBoost · LightGBM · Random Forest · CatBoost) across 10 prediction targets · evaluated on the 2025 backtest (21,753 events) at real-world prevalence

How to read this report

Each target is predicted by all 4 model families.

  • Brier — accuracy of the predicted probability (lower is better). Under Path B these are raw model probabilities, no post-hoc calibration.
  • ECE — calibration error: how closely "70% chance" matches reality (lower is better).
  • ROC-AUC / AP — ranking quality: can the model tell risky events from safe ones (higher is better, max 1.0).
  • R² / MAE / RMSE (regression) — variance explained and error magnitude; RMSE punishes large misses most.
  • Pick = weighted score — per target, each metric is min-max normalized across the four families (1 = best) then weighted. Classification: Brier 50% · ECE 25% · AP 15% · ROC 10%. Regression: RMSE 50% · MAE 30% · R² 20%.
  • Ensemble Pick — highlighted in green. For all classification targets, a single multi-class model (aftershock_spatial_zone_24h) must be served to guarantee disjoint probabilities. CatBoost was selected overall as it had the highest summed classification score across all 5 targets.
  • Green cell = the chosen ensemble model for that target (right column).
Every family is trained at natural prevalence (Path B — no post-hoc calibration) and scored on the 2025 backtest through the production inference path. The highlighted family per target is the chosen ensemble model, and this selection is the deployed mapping (kept in sync with HYBRID_MODEL_MAPPING in src/seis/predict.py).
