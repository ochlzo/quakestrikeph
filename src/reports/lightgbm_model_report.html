QuakeStrikePH LightGBM Aftershock Model

Summary of the current LightGBM model trained on the mc_1_0 labeled earthquake dataset, including production-style inference backtests using only historical data available before each event.

Current status: strong prototype, not production-ready yet
Report date
May 30, 2026

Source branch artifact
src/reports

Executive Read

Aftershock likelihood

Strong. The 24-hour yes/no classifier performs consistently in both offline testing and production-style backtesting.

Distance-bin probabilities

Useful as risk hints. Precision is high across bins, but recall varies, so the model is conservative.

Maximum magnitude

Experimental. The regressor improved after local seismic-history features, but still should not be used as a hard forecast.

What The Model Predicts

Output 1
24h
Probability that at least one aftershock occurs within 24 hours.
Output 2
6 bins
Probability of aftershock occurrence in radius bands from 0-10 km through 200+ km.
Output 3
M
Estimated maximum aftershock magnitude if an aftershock occurs within the forecast window.

Readiness Summary

Component Status Evidence Recommendation
24h likelihood classifier Promising ROC-AUC about 0.98 in production-style backtest. Proceed to threshold selection and probability calibration.
Distance-bin classifiers Useful High precision across all bins; recall varies by bin. Use as probabilistic hints, not exact radius guarantees.
Maximum magnitude regressor Experimental Backtest MAE about 0.91 magnitude units; R2 near zero. Improve before production use.
Production pipeline Prototype Inference and sampled backtest scripts exist. Add full-run backtesting, calibration, monitoring, and retraining flow.
